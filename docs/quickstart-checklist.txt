CHECKLIST RÁPIDO (LEVANTAR SIN PEDOS)

1) Docker sin sudo:
sudo usermod -aG docker $USER
newgrp docker
docker ps

2) Variables (NO se sube):
cp env/core.env.example .env
nano .env
- Cambia POSTGRES_PASSWORD
- Cambia N8N_ENCRYPTION_KEY (32+ chars)
- Si usarás proxy HTTP: N8N_SECURE_COOKIE=false

3) Levantar (elige receta):
A) Sin proxy (http://IP:5678)
./scripts/stack.sh up basic
./scripts/stack.sh ps basic

B) Con proxy (http://IP/)
./scripts/stack.sh up assistant
./scripts/stack.sh ps assistant

4) Si n8n truena por EACCES:
./scripts/stack.sh down basic
sudo chown -R 1000:1000 ./data/n8n
./scripts/stack.sh up basic

5) Permisos correctos Postgres:
docker exec lego-postgres sh -c 'id -u postgres; id -g postgres'
sudo chown -R <UID>:<GID> ./data/postgres

Si luego no puedes hacer ls:
sudo apt install -y acl
sudo setfacl -R -m u:$USER:rx ./data/postgres
sudo setfacl -R -d -m u:$USER:rx ./data/postgres
