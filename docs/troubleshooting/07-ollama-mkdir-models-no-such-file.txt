PROBLEMA: Ollama no puede crear /root/.ollama/models
SÍNTOMA:
Error: 400 Bad Request: mkdir /root/.ollama/models: no such file or directory: ensure path elements are traversable

CAUSA:
El volumen bind-mount ./data/ollama:/root/.ollama no tenía la carpeta models/ o tenía permisos que no permitían crear directorios.

SOLUCIÓN:
1) Crear estructura:
mkdir -p ./data/ollama/models

2) Permisos (recomendado):
sudo chown -R root:root ./data/ollama
sudo chmod -R 755 ./data/ollama

3) Reiniciar receta:
./scripts/stack.sh down assistant-ollama
./scripts/stack.sh up assistant-ollama

4) Descargar modelo:
docker exec -it lego-ollama ollama pull llama3.1:8b
docker exec -it lego-ollama ollama list
