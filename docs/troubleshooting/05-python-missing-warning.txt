PROBLEMA: Warning de Python missing en logs de n8n
SÍNTOMA:
"Failed to start Python task runner... because Python 3 is missing..."

SIGNIFICA:
n8n intentó arrancar un runner interno de Python (más para debug). En la mayoría de casos NO rompe n8n para flujos normales.

QUÉ HACER:
- Si n8n está corriendo bien, ignóralo al inicio.
- Si algún día necesitas tareas Python en serio, se recomienda runner externo (modo pro) en vez de meter Python dentro del contenedor.
