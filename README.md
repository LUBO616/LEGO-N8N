# LEGO-N8N 🧱🤖

Plantilla tipo LEGO para proyectos con **n8n + Docker Compose**.
- **Core:** n8n + Postgres
- **Módulos:** proxy, ollama, qdrant, minio, redis, gotenberg, etc.
- **Recipes:** combinaciones listas (ej. basic, assistant-hr)

## Quick Start (más adelante)
1. Copia variables: `cp .env.example .env`
2. Levanta receta: `./scripts/stack.sh up basic`

> Este repo está pensado para que actives/desactives módulos como piezas LEGO.
